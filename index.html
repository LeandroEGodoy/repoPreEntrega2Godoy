RagnaClub Online

To build a classic game compatible with modern gameplay, we created RagnaClub. In it, we strived to preserve the essence of the original game, interweaved with the comfort of a modern one. For this reason, we edited some ways in which the player interacts with the game, we configured some aspects of the game to give them their relevance back. We also added a variety of settings that allow you to customize the interaction with the various systems and NPCs, always prioritizing not to affect the gameplay and balance we all expect from a pre-renewal server. We love this game as much as you do, so we want to keep alive the nostalgia this game always causes, thus it is designed with great care. It is progressive to help you remember the old days of this wonderful game that is already a little over 20 years old.
